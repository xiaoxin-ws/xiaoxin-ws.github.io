js正则表达式学习笔记

xiaoxin-ws · 2019-8-19 · 次阅读

介绍

正则表达式（英语：Regular Expression，在代码中常简写为 regex、regexp 或 RE）使用单个字符串来描述、匹配一系列符合某个句法规则的字符串搜索模式。
搜索模式可用于文本搜索和文本替换。

什么是正则表达式？

  • 正则表达式是由一个字符序列形成的搜索模式。
  • 当你在文本中搜索数据时，你可以用搜索模式来描述你要查询的内容。
  • 正则表达式可以是一个简单的字符，或一个更复杂的模式。
  • 正则表达式可用于所有文本搜索和文本替换的操作。

使用

在 JavaScript 中，RegExp 对象是一个预定义了属性和方法的正则表达式对象。

创建正则

var regexp = new RegExp(/123/);//对象方式
var str1 = /123/;//字面量

test()使用

检查字符串是否符合正则

var rg = str.test('123');//true
var rg1 = str.test('wqeqw');//false
console.log(rg);
console.log(rg1);

exec()使用

  1. 检索字符串中指定的值。返回找到的值，并确定其位置。返回一个数组，没有找到，返回 null
/w/.exec(qwerrrtrtrydfgdw);

toString()使用

  • 返回正则表达式的字符串
function myFunction() {
    var patt = new RegExp("RUNOOB", "g");
    var res = patt.toString();
    document.getElementById("demo").innerHTML = res;
}

特殊字符（元字符）

包含

var str1 = /123/;//包含123

边界符 ^和$

var str11 = /^abc/;//以abc开头
var str12 = /^abc$/;//必须是abc
var str0 = /^abc/;//以abc开头
var str1 = /^abc$/;//必须是abc
var str2 = /[abc]/;//只要包含abc其中一个，就是true
var str3 = /^[abc]$/;//abc三选一，只是一个字母
var str4 = /[a-z]/;//只要包含26个小写字母中的一个
var str5 = /^[A-Z]$/;//26个大写字母中的一个
var str6 = /^[0-9]$/;//只有一个数字
var str7 = /^[a-zA-Z0-9]$/;//只有一个小写或大写字母或一个数字
var str8 = /^[^a-zA-Z0-9]$/;//[^],表示取反，

包含符 []

var str1 = /[abc]/;//只要包含abc其中一个，就是true
var str2 = /^[abc]$/;//abc三选一，只是一个字母
var str3 = /[a-z]/;//只要包含26个小写字母中的一个
var str4 = /^[A-Z]$/;//26个大写字母中的一个
var str5 = /^[0-9]$/;//只有一个数字
var str6 = /^[a-zA-Z0-9]$/;//只有一个小写或大写字母或一个数字
var str7 = /^[^a-zA-Z0-9]$/;//[^],表示取反，

量字符 * + ？ {3} {3,} {3,16}

var str0 = /^a*$/;//a可以重复0次或很多次
var str1 = /^a+$/;//a可以重复1次或很多次
var str2 = /^a?$/;//a可以重复1次或0次, 1||0
var str2 = /^a{3}$/;//a可以重复3次
var str2 = /^a{3,}$/;//a可以重复大于等于3次
var str2 = /^a{3,16}$/;//a可以重复3到16次
var str7 = /^[a-zA-Z0-9]{6,16}$/;//英文数字6-16位
var str8 = /^abc{3}/;//c重复3次

优先计算符 ()

var str8 = /^(abc){3}/;//abc重复3次

正则表达式的修饰符

  • i 不区分大小写
  • g 执行全局匹配（查找所有匹配，而不是在找到第一个后停止）
  • m 执行多行匹配

预定义类

  1. \d 匹配 0-9 之间的任一数字，相当于[0-9]
  2. \D 匹配 0-9 以外的字符，相当于[^0-9]
  3. \w 匹配任意的字母，数字和下划线，相当于[A-Za-z0-9_]
  4. \W 匹配任意的非字母，非数字和非下划线，相当于[^a-za-z0-9_]
  5. \s 匹配空格（包含换行符，制表符，空格符等）相当于[\t\r\n\v\f]
  6. \S 匹配非空格（包含换行符，制表符，空格符等）相当于[^\t\r\n\v\f]
  7. \b 匹配单词的边界
  8. \uxxxx 查找，以 16 进制数 xxxx 规定的 Unicode 字符

一些例子

var  objRegExp= /^\d+\.\d+$/;

校验是否中文名称组成

var reg=/^[0-9]{8}$/;

校验邮件地址是否合法

  var reg=/^\w+@[a-zA-Z0-9]{2,10}(?:\.[a-z]{2,4}){1,3}$/;

一个单词连续出现的位置。

  var reg=/\b([a-z]+) \1\b/gi;

将一个 URL 解析为协议、域、端口及相对路径。

  var reg=/(\w+):\/\/([^/:]+)(:\d*)?([^# ]*)/;

定位章节的位置。

  var reg=/^(?:Chapter|Section) [1-9][0-9]{0,1}$/;

匹配空行。

  var reg=/<\s*(\S+)(\s[^>]*)?>[\s\S]*<\s*\/\1\s*>/;

前端开发常用的表达式

  • 原文: http://www.imooc.com/article/290261?block_id=tuijian_wz

1、 手机号：mobile

/^1((3[\d])|(4[5,6,9])|(5[0-3,5-9])|(6[5-7])|(7[0-8])|(8[1-3,5-8])|(9[1,8,9]))\d{8}$/

2、国内座机电话（如: 010-12345678）：telephone

/\d{3}-\d{8}|\d{4}-\d{7}/

3、电话或手机：phoneNo

 /(^1([3|4|5|7|8|])\d{9}$)|(^(\(\d{3,4}\)|\d{3,4}-|\s)?\d{7,14}$)/

4、email 地址：email

/^\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*$/

5、身份证：IDcard

 /^[1-9]\d{7}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{3}$|^[1-9]\d{5}[1-9]\d{3}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{3}([0-9]|X)$/

6.账号 4-10 位数字或字母组成

/^[0-9A-Za-z]{4,10}$/

8、账号 4 到 16 位（字母，数字，下划线，减号）

/^[a-zA-Z0-9_-]{4,16}$/

9.数字或字母或汉字：numAlphaCn

/^[0-9a-zA-Z\u4E00-\uFA29]*$/

10.中文和数字组成

/^(([\u4E00-\u9FA5])|(\d))+$/

11.是否为中文

/^[\u4E00-\u9FA5]/

12.整数：int

/^-?\d+$/

13.域名

[a-zA-Z0-9][-a-zA-Z0-9]{0,62}(/.[a-zA-Z0-9][-a-zA-Z0-9]{0,62})+/.?

14.网址

[a-zA-z]+://[^\s]* 或 ^http://([\w-]+\.)+[\w-]+(/[\w-./?%&=]*)?$

15.整数且大于 0：intAndThanZero

/^([1-9]\d*(\.\d+)?|0)$/

16.小数

/^\d+\.\d+$/

17. 8 位纯数字

/^[0-9]{8}$/

18. 正整数、小数或 0

/^\d+(\.?|(\.\d+)?)$/

19. 上传图片类型

/image\/(png|jpg|jpeg|gif)$/

20. 日期格式,如: 2000-01-01

/^\d{4}(-)\d{1,2}\1\d{1,2}$/

21.一年的 12 个月(01 ～ 09 和 1 ～ 12)：

/^(0?[1-9]|1[0-2])$/

22.一个月的 31 天(01 ～ 09 和 1 ～ 31)：

/^((0?[1-9])|((1|2)[0-9])|30|31)$/

23.是否 html 标签

/<(.*)>.*<\/\1>|<(.*) \/>/

24.是否是 qq 号

/^[1-9]*[1-9][0-9]*$/

25. 密码强度正则，最少 6 位，包括至少 1 个大写字母，1 个小写字母，1 个数字，1 个特殊字符

/^.*(?=.{6,})(?=.*\d)(?=.*[A-Z])(?=.*[a-z])(?=.*[!@#$%^&*? ]).*$/

26. ipv4 地址正则

/^(?:(?:25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\.){3}(?:25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)$/

27. 16 进制颜色

/^#?([a-fA-F0-9]{6}|[a-fA-F0-9]{3})$/

28. 6 至 20 位，以字母开头，字母，数字，减号，下划线（微信号）

/^[a-zA-Z]([-_a-zA-Z0-9]{5,19})+$/

29.首尾空白字符

^\s*|\s*$或(^\s*)|(\s*$)

30.邮政编码：postCode

/^(0[1-7]|1[0-356]|2[0-7]|3[0-6]|4[0-7]|5[1-7]|6[1-7]|7[0-5]|8[013-6])\d{4}$/
